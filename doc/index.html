GET /hello_world

This service provides a simple hello world implementation example.

  • curl -I 'http://localhost:9292/hello_world?name=Matt'

Input parameters description

Optional Params

  • name of type string The name of the person to greet.  Default Value: World 

Response description

Properties:
  • message of type string The greeting message sent back. Defaults to 'World'
  • at of type datetime The timestamp of when the message was dispatched

Response example

{"message":"string","at":"datetime"}

POST /users/:username

This service provides a simple hello world implementation example.

  • curl -I 'http://localhost:9292/hello_world?name=Matt'

Input parameters description

Required Params

  • username of type string The name of the person to greet. 

Response description

Properties:
  • login of type string The name of the person to greet.
  • location of type string The location of the person.
  • flags of type Array
    Properties of each array item:
    • name of type string The name of the person to greet.
    • description of type string The name of the person to greet.
    • owner of type string The name of the person to greet.
    • language of type string The name of the person to greet.

Response example

{"login":"string","location":"string"}

GET /users/:username

This service provides some information of the person and projects flagged by him.

  • curl -I 'http://localhost:9292/users/vincentv'

Input parameters description

Required Params

  • username of type string The name of the person to find. 

Response description

Properties:
  • login of type string The nickname of the person.
  • location of type string The location of the person.
  • flags of type Array
    Properties of each array item:
    • name of type string The name of the project.
    • description of type string The description of the project.
    • owner of type string The owner's nickname of the project.
    • language of type string The language used in the project.

Response example

{"login":"string","location":"string"}
